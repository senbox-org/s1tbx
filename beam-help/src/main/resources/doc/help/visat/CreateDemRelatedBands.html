Create DEM-related Bands

This command adds a new DEM-elevation band and optionally DEM-corrected latitude and longitude bands to the current product.
The bands are computed by looking up each pixel in a selected high-resolution DEM (which is by default GETASSE30). In opposite to the orthorectification tool, which always reprojects the data product, the DEM-corrected latitudes and longitudes are computed in the selected product's current coordinate system.

  • Digital elevation model (DEM)
    Lets you select a DEM from the list of currently installed DEMs.
  • Elevation band name
    The name of the new elevation band.
  • Output DEM-corrected bands
    Check if DEM-corrected bands shall be created. Depending on the data product type, this option may not be available.
  • Latitude band name
    The name of the new DEM-corrected latitude band.
  • Longitude band name
    The name of the new DEM-corrected longitude band.
