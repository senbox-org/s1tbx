Create Geo-Coding Displacement Bands

This command adds 4 bands to the currently selected product. These four bands provide information about the accuracy of the geo-coding. Note, that this doesn't mean the absolute accuracy of the geo-coordinates, but only the transformation from pixel positions to geo-coordinates and back. The four created bands are namely gc_displ_x, gc_displ_y, gc_displ_ampl and gc_displ_phase.
