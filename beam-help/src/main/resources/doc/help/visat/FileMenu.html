File Menu

File Menu

New Product  

The new-command allows you to create a new product. The command is enabled only if you have already opened or imported a product from which the new product can derive it's geo-coding information. The primary use of this command is to create a spectral or spatial subset of another, open product.

New Time Series  

The command allows the creation of a time-series product. Multiple source products can build up a time-series. All source must share a common coordinate reference system (CRS). If this is not the case the products are reprojected on th fly.

Open Product (Control-O)  

Opens one or multiple data products stored on the hard disk in the Sentinel Toolbox. You can select more than one product, e.g. by pressing the CTRL and select the desired products with the mouse.

Open Session (Control-Shift-O)

Opens a SNAP Session file stored on the hard disk in the Sentinel Toolbox. A session is termed the collection of all products and image views open in the Sentinel Toolbox, including any layers added to a view. When a session file is opened, a previously saved session (see Save Session) is restored. Note that all open products will be closed before a session file is opened.

Reopen  

The Sentinel Toolbox stores the location of files previously opened or saved in this menu group. Select an entry and the Toolbox will re-open the product.

OPeNDAP Access  

An OPeNDAP client for accessing a OPeNDAP server is shown a new dialogue. The provided data products of the server can be downloaded to the hard disk and opened in the Sentinel Toolbox.

Product Grabber  

Opens the Product Grabber for data product previewing, opening or importing reasons.

Close Product (Control-W)

Closes the currently selected product. This commands is also available in the context menu when you right-click over the open product list.

Close Session (Control-Shift-W)

Closes all products and image views currently opened in the Sentinel Toolbox. Note that changes made during a session are not written to an associated SNAP Session file. In order to write the changes to a session file the session must be saved (see Save Session) explicitly.

 

Save Product (Control-S)  

Saves the currently selected product in the BEAM-DIMAP format. If the product still does not have a file location, then you are asked to specify a storage location for the product. Note that save process can be a time consuming process for huge products.

Save Product As  

Saves the currently selected product to a different location. You are asked to specify a new storage location for the selected product.

Save Session (Control-Shift-S)

Saves the current Session to the associated SNAP session file. If the current session is not associated with a SNAP session file, you are asked to specify a storage location for the session file.

Save Session As

Saves the current session to a new SNAP session file. You are asked to specify a storage location for the new session file.

Import Raster Data

Allows the user to import raster data from multiple formats. See help for Sentinel Toolbox -> Tools -> Import and Export Tools -> Import Raster Data for detailed information.

Import Vector Data

Allows the user to import vector data from multiple formats and to interpret it in multiple different ways. See help for Sentinel Toolbox -> Tools -> Import and Export Tools -> Import Vector Data for detailed information.

Export Raster Data

Allows the user to export raster data into multiple formats. See help for Sentinel Toolbox -> Tools -> Import and Export Tools -> Export Raster Data for detailed information.

Other Exports

Provides multiple different exports, such as to export the view as image. See help for Sentinel Toolbox -> Tools -> Import and Export Tools -> Other Exports for detailed information.

Exit

Automatically stores all preferences, last visited directories frame size and location and then exits the Sentinel Toolbox application.
