Import MERIS Binned Level-3 Product

This option allows to import MERIS Binned Level-3 data products into VISAT. These data products provide the Level-3 statistics mean, count, min and max for a single MERIS Level-2 variable, e.g. Chlorophyll-a on a global, 9 km, integerised sinusoidal (ISIN) grid. The statistics for each variable is provided in a single netCDF file.

For the ease of use, BEAM resamples the global ISIN grid to a global Plate Carrée grid (Lat/lon projection).

MERIS Binned Level-3 data products can be freely downloaded from the MERIS Level-3 Products Website (http://www.enviport.org/meris/lv3_main.htm). Please copy the address and use your web-browser (e.g. Internet Explorer) to download the products. The download will not work with the browser included in this help-system. A description of the file format and the ISIN grid can also be found there.
