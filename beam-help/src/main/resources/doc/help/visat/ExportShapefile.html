Export Geometry as Shapefile

This command allows the export of geometries as ESRI Shapefile.

When exporting a geometry with different geometry types (Line, Polygone, etc.), than a separate file for each type will be created. The type name will be appended to the filename you have entered. This results in:

  • <MyShapeFile>_Line.shp
  • <MyShapeFile>_Polygone.shp
  • etc.
Also multiple associated files will be created for each shapefile: *.dbf, *.fix, *.shx, *.prj
