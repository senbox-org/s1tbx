Import NetCDF/CF

The NetCDF/CF reader enables VISAT to import generic NetCDF/CF data products.

The NetCDF/CF reader supports any image-like NetCDF/CF file structure up to NetCDF version 3.5. The currently used NetCDF Java library version is 4.0.46.

Note: You can find additional information about NetCDF at http://www.unidata.ucar.edu/software/netcdf/ and about the NetCDF Climate and Forecast (CF) Metadata Convention at http://cf-pcmdi.llnl.gov/.

Additionally the NetCDF library also handles reading of HDF4 and HDF5 files. Currently it supports reading of HDF-EOS products in gridded format using the Lat/Lon projection and the sinusoidal projection. These are e.g the MODIS L3 products.
