Import MERMAID Extraction Files

BEAM/Visat supports the import of MERMAID (MERis MAtchup In-situ Database) Extraction Files, which have become a very important data source for merged in-situ optical measurements with concurrent MERIS acquisitions. In general, these files follow the BEAM CSV import and the underlying rules, with just a few exceptions (i.e. semicolon as separation character, special latitude/longitude column names.

A comprehensive description of the MERMAID project and details on the matchup extraction data format can be found here.
