Help Menu

The Sentinel-3 Toolbox Window Menu contains:

  •   Help Topics
    Opens the Sentinel-3 Toolbox help.
  • Sentinel Home Page
    Opens the Sentinel web site. https://sentinel.esa.int/web/sentinel/home
  • Access to Data Products
    Shows an information page of data product distributors.
  •   Module Manager...
    The Module Manager for updating, installing and unstalling modules.
  • Check for New Release...
    Checks for a new Sentinel-3 Toolbox release.
  •   About SNAP
    Opens the 'About SNAP' window.
