Create ASAR NRCS Bands

This command adds a two new NRCS (Normalised Radar Cross Section) bands to an ENVISAT ASAR product:

  1. nrcs = (proc_data * proc_data / ext_cal_fact) sin(rad(incident_angle)).
  2. nrcs_db = 10 * log10(nrcs) which provides the NRCS in the more common dB units.

In these expressions proc_data are the ASAR image pixel counts, ext_cal_fact is a calibration factor taken from the product's metadata (MAIN_PROCESSING_PARAMS_ADS) and incident_angle is the tie-point grid providing the incident angle in degrees.

The command is only active, if all information required to compute the NRCS can be retrieved from a selected ASAR product.
