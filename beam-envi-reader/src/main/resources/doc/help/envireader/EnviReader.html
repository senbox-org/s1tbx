Envi Data Product Reader

Envi Reader

The reader supports data products in the Envi data format. The products consist of pairs of a header (.hdr) and an image file (.img, .bin, .bil or .bsq). The data can also be read from a zip file.

HICO Reader

The data of the HICO sensor (http://www.sdl.usu.edu/programs/hico) is delivered in the Envi format. Beside the header and image file HICO products come along with other ancillary files like RGB, NDVI and flag file.
