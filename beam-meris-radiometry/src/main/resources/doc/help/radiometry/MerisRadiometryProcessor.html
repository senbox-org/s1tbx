MERIS Level 1 Radiometry Processor

Introduction

The MERIS Level 1 Radiometry Processor offers the ability to preprocessor MERIS L1 data products. One or all of the following algorithms can be performed on the MERIS Level 1 source product:

  • SMILE Correction
  • Equalization
  • Radiometric Recalibration
  • Radiance to Reflectance Conversion
