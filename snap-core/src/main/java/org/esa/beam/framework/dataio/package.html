This package is defines the MERIS/(A)ATSR Toolbox product I/O framework. The product I/O framework is used to dynamically integrate new product readers and writers for different data formats into the MERIS/(A)ATSR Toolbox. A product reader or writer plug-ins implements the {@link org.esa.beam.framework.dataio.ProductReaderPlugIn} resp. {@link org.esa.beam.framework.dataio.ProductWriterPlugIn} interface.
