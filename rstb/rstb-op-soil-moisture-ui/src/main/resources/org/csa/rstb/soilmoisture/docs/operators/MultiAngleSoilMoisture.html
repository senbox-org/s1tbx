Multi-Angle IEM Soil Moisture Retrieval


By taking a two-step approach, soil moisture (SM) can be retrieved from radar backscatter coefficients.

Step 1: Use IEM Multi-Angle inversion to obtain the Real Dielectric Constant (RDC) from radar backscatter coefficients.

Step 2: Invert SM dielectric model to retrieve SM from RDC.

IEM Multi-Angle Inversion

Source product must contain both AM and PM images.

The Multi-Angle approach utiliizes one backscatter coefficient (HH or VV) from each of the two images (AM or PM).  Use can select one of the four possible polarization combinations.

User can also browse for a calibrated IEM LUT. Matlab (.mat) and CSV (.csv) formats are supported.

SM Dielectric Model

User can choose one of two models: Hallikainen or Mironov.

Sand and clay percentages are required as input for Hallikainen while Mironov only requires clay percentage. User can choose a resampling method to generate the bands and add them to the source product.

For both models, the user can enter the SM range (minimum and maximum) for retrieval and optionally select to output RDC as well as the land cover bands.

Finally, for Mironov model only, the user needs to enter the effective soil temperature.
