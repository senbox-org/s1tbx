No-Data Overlay

   No-Data Overlay

Toggles the overlay of a no-data mask within a band's image view. The overlay properties can be modified using the no-data overlay page in the preferences dialog.

The no-data mask is similar to a bitmask overlay and masks out all the no-data pixels in the selected image view.
If the property no-data value of a band or the valid pixel expression of a band is set, a no-data overlay can be displayed within the current image view.
