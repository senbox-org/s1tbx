Import ImageIO

Import ImageIO Product

This option allows to import any format supported by Java ImageIO plug-ins.

The built in ImageIO readers/writers include:

  • BMP
  • PNG
  • GIF
  • JPEG
  • BPM
  • PPM
  • PGM
  • RAW

Abstracted Metadata can be imported by creating a metadata.xml file in the same folder as the image.
