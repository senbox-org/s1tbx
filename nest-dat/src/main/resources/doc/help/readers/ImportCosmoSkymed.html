Import Cosmo-Skymed

Import Cosmo-Skymed Product

This option allows to import Cosmo-Skymed products into DAT.

The Cosmo-Skymed product reader supports SCS, DGM, GEC for the following modes:

  • ScanSAR Huge Region (DGM, GEC only) 
  • ScanSAR Wide Region 
  • Spotlight Enhanced 
  • Stripmap Himage
  • Stripmap PingPong

Currently the SCS ScanSAR Huge Region is not supported.
