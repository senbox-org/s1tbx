Import ACE30 DEM

Import ACE30 DEM Tile

Imports a tile of the ACE elevation model at 30 arc second resolution referenced to the EGM96 geoid.

See ACE Report at http://www.cse.dmu.ac.uk/geomatics/products_ace_overview.html

Since the ACE height information is referred to geoid EGM96, not WGS84 ellipsoid, correction has been applied to obtain height relative to the WGS84 ellipsoid.
