Export ROI Overlay

If the active image view has a ROI overlay, this command will be enabled. It allows the export of the current ROI overlay as an image file.

The exported image has the same pixel size as the active image, with the ROI marked in the selected overlay colour.

The (presently) supported formats are:

  • BMP: Microsoft Windows Bitmap (*.bmp)
  • PNG: Portable Network Graphics (*.png)
  • JPEG: Joint Photographic Experts Group (*.jpg, *.jpeg)
  • TIFF: Tagged Image File Format (*.tif, *.tiff)
  • GeoTIFF: TIFF with geo-location (*.tif, *.tiff)

Note: From these image formats only PNG supports transparency.
