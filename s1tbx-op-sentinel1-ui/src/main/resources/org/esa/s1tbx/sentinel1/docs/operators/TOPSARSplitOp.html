TOPSAR Split

Sentinel-1 TOPSAR Split

The TOPSAR Split operator provides a convenient way to split each subswath into a separate product. This operator is the first processing step in the TOPS InSAR processing chain.

The user may select the desired subswath and polarization.

Input

  • The input to this operator should be Sentinel-1 multi-swath SLC product (e.g. IW product with VH and VV polarisations).

Output

  • The output of the operator is the split Sentinel-1 product with user selected subswath and polarisation(s).

Parameters Used

  • Subswath: The subswath for the split product.
  • Polarisation: The polarisation(s) for the split product.
