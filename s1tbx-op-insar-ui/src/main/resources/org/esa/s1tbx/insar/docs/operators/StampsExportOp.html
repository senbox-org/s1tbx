StaMPS Export

Use the StaMPS Export to produce data that can be used within the StaMPS applicaiton for Persistent Scattering Interferometry (PSI).

Inputs: 

  1. Coregistered SLC stack with a master iamge and multiple slave images.
  2. Interferogram for master/slave pairs and an elevation band

Apply precise orbit correction and coregister a stack of SLC products.

Generate interferograms for the stack.

Add an elevation band to the interferogram.


The StaMPS Export will take the coregistered stack and interferogram with elevation and produce an output in Gamm format.

You must then run the mt_prep_gamma script from StaMPS to prepare the Gamma formatted data.

Then run stamps in matlab.
