Horizontal/Vertical Motion

Horizontal/Vertical Motion Operator

Given two (collocated) ascending and descending interferograms, this operator computes the horizontal (East-West) and vertical motion. It is assumed that the motion occurred in these two directions only.

Input

The input to this operator should be a product that has resulted from collocating two terrain-corrected interferograms. Such a collocated product can be created by following these steps:

  • Create an interferogram for an ascending orbit.
  • Create an interferogram for a descending orbit (note: the dates of each InSAR pair should be selected and sorted such that they correspond with the ascending interferogram).
  • Multilook (or apply Goldstein filter to) both interferograms and perform phase unwrapping.
  • Apply the Range Doppler Terrain Correction operator to both interferograms, so that they are geocoded. Make sure you select the "Incidence angle from ellipsoid" option.
  • Use the Collocate operator to align the terrain-corrected interferograms.

Output

The output of this operator is a product with the following bands:

  • horizontalMotion: East-West deformation in millimeters.
  • verticalMotion: vertical deformation in millimeters.

Parameters

The X and Y positions of a reference pixel should be selected.
