Goldstein Phase Filtering

Goldstein Phase Filtering Operator

Phase filtering is a preprocessing technique that greatly reduces the residues in the later on  phase unwrapping step and enhances the phase unwrapping accuracy. The method implemented in this operator is a nonlinear adaptive algorithm proposed by Goldstein and Werner [1] in 1998.

Input and Output

  • The input to this operator is the output of Interferogram Formation operator.
  • The output of this operator is a stack of filtered I/Q bands.

Parameters Used

   The following processing parameter is required for this operator (see Figure 1):
  • Adaptive Filter Exponent: A value lies in range (0,1]. The larger the value the stronger the filtering.
