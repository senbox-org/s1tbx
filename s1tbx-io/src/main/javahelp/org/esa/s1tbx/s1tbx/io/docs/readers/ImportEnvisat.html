Import ENVISAT (ASAR, MERIS, AATSR) Product

The Advanced Synthetic Aperture Radar (ASAR) onboard the ENVISAT satellite was a global mapping SAR mission of ESA.

Operational: 2002-03-01 to 2012-04-08

Frequency Band: C-band

Acquisition Modes:

  • Image Mode (IM),
  • Alternating Polarization (AP),
  • Global Monitoring (GM),
  • Wide Swath (WS),
  • Wave (WV)

Polarizations: All modes are single pol except AP mode which offers non-coherent dual pol.

Opening products: Products can be opened by selecting the *.N1 file. Opening zipped products (*.zip) directly is also supported.
